## 什么是栈

- 栈又名为堆栈，栈是一种特殊的线性表结构。
- 它只允许在一端进行插入和删除操作。遵循后进先出（LIFO）的原则。
- 栈支持通过数组或链表实现，数组实现的叫做顺序栈，链表实现的叫链式栈。

## 栈的基本元素

- 栈主体
- 栈顶
- 栈底

## 空间、时间复杂度

- 空间复杂度：O(1)
    - 只需一个大小为 n 的数组
- 时间复杂度：O(1)
    - 入栈、出栈只涉及栈底或栈顶的操作
    - 数组实现自动扩容，个别情况可能是 O(n)

## 使用场景

- 逆序输出数据
- 数值转换
- 浏览器的前进与后退
- 编辑器/IDE 的撤销与取消撤销
- 编辑器/IDE 的语法检查
